## How to run backend :
- clone the backend repo :
    - `git clone https://github.com/daniarthurwidodo/responsive-registration-form-backend.git`
- go to repo folder
    - `cd responsive-registration-form-backend`
- install library
    - `npm install`
- run backend server
    - `npm run dev`
